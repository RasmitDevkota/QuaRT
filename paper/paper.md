# Summary

`QuaRT` is a Python library for quantum simulation of radiative transfer in astrophysical and cosmological problems.

The source code for `QuaRT` is available on [GitHub](https://github.com/RasmitDevkota/QuaRT/). It can be installed via `pip` from the [`pypi` index](https://pypi.org/project/TODO/). Its [documentation](https://quart.readthedocs.io/) is hosted publicly.
